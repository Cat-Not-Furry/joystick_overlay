# Índice (redirige)

El índice navegable de la documentación vive en **[README.md en esta carpeta](README.md)** (`docs/README.md`).

Usa ese fichero para enlaces por audiencia (usuarios, streamers, desarrollo, seguridad, referencia, archivo).

**Política:** este fichero se mantiene como **enlace estable** hacia el índice canónico; no duplicar aquí tablas largas. Si enlazas desde fuera del repo, puedes usar `docs/INDEX.md` o directamente `docs/README.md`.
